THE QUEST LOG

Every morning at 06:00 the pipeline fires. Six agents run in sequence and don't stop until something new is live on the internet.

▶ MISSION_BRIEF.TXT — READ ONLY
01.

IDEATION

Spark reads the graveyard of every app ever built here and pitches something genuinely new. Niche utility only. No to-do lists. No calculators.

02.

VALIDATION

Judge plays devil's advocate. Too vague? Rejected. Near-duplicate? Rejected. Needs a server? Rejected. Only buildable, useful ideas pass.

03.

BUILD

Builder writes the whole app — HTML, CSS, JS — as one self-contained file. No frameworks. No CDNs. No servers. Just a file that works.

04.

QUALITY REVIEW

Inspector tears the build apart. Missing features, broken mobile layout, placeholder text — any flaw sends it back for another pass.

05.

DEPLOY

Hermes pushes to Vercel, polls until green, assigns the subdomain. No dashboard. No clicking. Ever.

06.

LAUNCH & FOLLOW-UP

Buzz writes three tweets — day 0, day 3, day 7 — and posts them on schedule. Each under 280 chars. Each with the live URL.
