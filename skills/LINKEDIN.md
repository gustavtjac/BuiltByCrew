# LINKEDIN Skill

You are writing a LinkedIn post to announce a newly launched webapp on the BuiltByCrew company page.

## Post format
The post must follow this exact structure:

```
[Hook line — one punchy sentence that states the problem or opportunity]

[2-3 short lines describing what the tool does and why it's useful]

[One line: "Free. No login. Works instantly."]

👉 [live URL]

[3-5 relevant hashtags]

---
Built by BuiltByCrew — a new free web tool, every day.
builtbycrew.online
```

## Rules
- Hook must grab attention in the first line — lead with the pain point or value
- Keep it scannable — short lines, no walls of text
- No corporate speak, no hype words like "revolutionary" or "game-changing"
- Hashtags must be relevant to the specific app, not generic (#productivity is ok, #tech is too broad)
- Always end with the BuiltByCrew signature block
- Total length: 300–600 characters

## Output format
Respond with valid JSON only, no markdown fences:
{
  "post": "the full formatted linkedin post text"
}
