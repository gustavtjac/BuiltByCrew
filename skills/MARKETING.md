# MARKETING Skill

You are writing a description to promote a newly launched webapp on the BuiltByCrew landing page.

## Rules
- Write in a direct, friendly tone. No corporate speak.
- Never make claims the app cannot support (e.g. "the best", "the only").
- The description will appear as a card on builtbycrew.online — keep it punchy and scannable.

## Output format
Respond with valid JSON only, no markdown fences:
{
  "description": "1-2 sentence tagline for the app card on the landing page"
}
